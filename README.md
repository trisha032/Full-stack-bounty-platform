# Full-stack-bounty-platform
# 🏆 Bounty Platform (Full Stack dApp)

A decentralized full-stack bounty platform built using **Stellar Soroban smart contracts**, allowing users to create, manage, and complete tasks in a trustless environment.

---

## 🚀 Live Overview

The Bounty Platform enables:

* Users to post tasks with rewards 💰
* Contributors to complete tasks 👨‍💻
* Transparent and secure on-chain tracking 🔗

---

## 🧠 Project Description

This project is a **full-stack decentralized application (dApp)** that integrates blockchain technology with a modern web interface. It eliminates the need for intermediaries by using smart contracts for managing bounties and ensuring fairness between creators and contributors.

---

## ✨ Features

* 🧾 Create Bounty with reward
* 👨‍💻 Complete Bounty
* 🔐 Wallet-based authentication
* 🔍 On-chain transparency
* 📦 Decentralized data storage
* 🌐 Full-stack integration (Frontend + Smart Contract)
